Gestione dei backup da Utilità di pianificazione di Windows

La schedulazione dei backup attraverso le Utilità di pianificazione di Windows è necessaria solo con SQL Server 2005/2008 Express Edition dove non è disponibile l'SQL Server Agent.

L'utilizzo delle Utilità di pianificazione rende comunque più critico e vulnerabile il processo backup per il fatto che talvolta tali processi si possono bloccare e non vi sono efficaci notifiche di eventuali errori occorsi. Pertanto va valutata bene l'adozione di SQL Server 2005/2008 Express Edition: in installazione critiche è opportuno utilizzare SQL Server 2005/2008 Standard Edition.

Per creare una nuova attività in Utilità di pianificazione (da Pannello di controllo > Strumenti di amministrazione) si esegue, dalla colonna Azioni, Crea attività.

Verificate che l'utente utilizzato per eseguire l'attività pianificata disponga degli accessi per eseguire le azioni richieste, altrimenti utilizzate un utente diverso con Cambia utente o gruppo.

Ricordarsi di selezionare l'opzione Esegui indipendentemente dalla connessione degli utenti per consentire l'esecuzione anche ad utente disconnesso.

Si passa poi ad Attivazione per schedulare l'attività di backup SQL.

Da Nuovo si accede alla definizione del tipo di pianificazione che deve essere minimo settimanale, nei giorni feriali di utilizzo effettivo di Business.

Si passa poi ad Azioni per specificare l'azione da eseguire, nel nostro caso si tratta del comando di backup, prodotto da Business in Gestione Aziende/Database, Schedala Backup, col pulsante Genera file BAT:

Tale file BAT include sia il comando per il backup del database completo seguito, se applicabile, dal comando per eseguire il backup del log delle transazioni. Per evitare di dover schedulare più attività si possono raggruppare in un unico file BAT tutti i comandi SQLCMD di backup dei database di Business.

A conferma dell'inserimento della nuova attività viene richiesta la password dell'utente adibito alla sua esecuzione.

A questo punto la nuova attività è stata inserita e l'Utilità di pianificazione visualizza il riepilogo della nuova schedulazione.
