Impostazione dati esterni DWH

Nome programma: BNDWIMPE.DLL
Moduli richiesti: Datawarehouse Extended

Questo programma definisce fact/dimensioni da fonti di dati esterne, provenienti da Excel, Access o base dati Sql Server, utilizzate nel programma di Analisi dati Datawarehouse. Dopo l'apertura del programma, occorre selezionare un preciso scenario sul quale andremo a lavorare e a definire i relativi fact/dimensioni. Vedere anche:

  • Integrazione del datawarehouse con fonti dati esterni (SQL Server)
  • Integrazione del datawarehouse con fonti dati esterni (Excel)
  • Integrazione del datawarehouse con fonti dati esterni (Access)

Progr. import Digitare un numero che rappresenta il codice relativo ai dati esterni.

Tipo estensione Indicare il tipo di fonte dati esterno tra Excel, Access e Sql Server.

Percorso file Excel o Access Inserire il percorso completo al file Excel o Access, comprensivo di nome ed estensione del file; ad esempio, C:\Bus\Asc\Prova.xls.

Pagina Excel o Nome tabella É il nome della pagina nel file di Excel o il nome della tabella di Access o Sql Server.

Descrizione É la descrizione relativa al tipo d'impostazione dei dati esterni.

Riga iniziale Excel É il numero della riga inziale relativo alla griglia dei dati.

Versione Excel Access Impostare la versione di Excel o Access a seconda del tipo di file utilizzato.

Stringa di connessione Inserire la stringa in sintassi ODBC di connessione al database Sql Server.

Impostazione dati esterni DWH Dimensione Fact

Per ogni riga inserita precedentemente, tramite apposita voce di menù occorre richiamare una finestra che consente d'inserire le impostazioni relative ad una configurazione di impostazione dati esterni.

Progr. campo Digitare un numero che identifica il codice relativo ai fact/dimensione campo zoomabile.

Des. Progr. campo Riporta la decodifica della descrizione del codice numerico relativo al fact/dimensione.

N. riga Inserire il progressivo per elaborazioni di colonne multiple altrimenti lasciare a zero.

Ordinamento Ordinamento in fase di elaborazione, per ora non utilizzato.

Tipo cella Il tipo di valore collegato al fact/dimensione: Costante, Cella unica fissa, Colonna, Colonna multipla e Calcolato dai suoi componenti.

Valore costante Il valore costante da impostare nel fact/dimensione nel caso Costante.

Id cella L'id della cella di Excel da impostare nel fact/dimensione nel caso Cella unica fissa.

Id colonna o nome campo L'id della colonna di Excel da impostare nel fact/dimensione nel caso Colonna.

Progr campo1 / Progr campo2 / Progr campo3 Il progressivo del fact/dimensione che dipende dalla colonna multipla di Excel per il per il caso Colonna multipla.

Des. Progr campo1 La descrizione del fact/dimensione relativo al campo1.

Valore costante1 Il valore costante relativo relativo al campo1.

Id cella1 Il valore della cella di Excel relativo al campo1.

Des. Progr campo2 La descrizione del fact/dimensione relativo al campo2.

Valore costante2 Il valore costante relativo relativo al campo2.

Id cella2 Il valore della cella di Excel relativo al campo2.

Des. Progr campo3 La descrizione del fact/dimensione relativo al campo3.

Valore costante3 Il valore costante relativo relativo al campo3.

Id cella3 Il valore della cella di Excel relativo al campo3.
