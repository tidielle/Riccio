Stampa elenchi Intrastat

Nome programma: BSICSTTR.DLL
Moduli richiesti: Intrastat
Opzioni programma: vai alle opzioni
Report programma: vai ai report

Questo programma effettua un'elaborazione, in modalità di prova o definitiva, degli elenchi Intrastat da presentare in dogana: viene generato un file Ascii da presentare in dogana, insieme al frontespizio.

Contestualmente è generata una serie di stampe da intendersi ad uso interno, a fini di archivio o controllo: non sono stampe da presentare in dogana. Le stampe relative ai servizi bsic1qua.rpt, bsic1qui.rpt, bsic2qua.rpt, bsic2qui.rpt si basano sulla relativa tabella temporanea TTSTINTRSER.

L'elaborazione avviene secondo alcune prerogative:

- tutte le righe di rettifica relative a periodi precedenti devono avere riferimenti a righe di elenchi da stornare;

- nei movimenti intra è memorizzato il progressivo sez. 3, perciò l'aggiornamento del progressivo sui movimenti avviene durante la generazione del file scambi.cee;

- sono riportate sempre informazioni sia fiscali che statistiche, anche nei casi in cui alcune delle informazioni non siano richieste;

- per identificare i record a fini statistici, Business verifica che i campi "Importo" e "Importo in valuta" siano pari a 0: in caso affermativo, sul file ascii non sono riportati il paese e l'identificativo del cliente/fornitore estero, perchè in questo caso non sono richiesti. Questo comportamento è regolato dall'opzione di registro BSICSTTR/OPZIONI/RigheSoloStatisticheImpZero: se pari a -1 (valore di default), considera le righe Intrastat aventi importo = 0 a fini statistici; se pari a 0, esporta in ogni caso su file il valore dei due campi sopra indicati.

Si consiglia di controllare sempre il contenuto del file prima della presentazione, utilizzando l'apposito programma "Intr@web" fornito gratuitamente dagli uffici doganali.

Tipo

Estrai Selezionare il tipo di estrazione da effettuare: solo merci, solo servizi o entrambi, considerando cessioni/servizi resi o acquisti/servizi ricevuti.

Periodicità presentazione

Anno/mese/trimestre Imputare il mese o trimestre, a seconda della periodicità di presentazione degli elenchi, e l'anno di riferimento.

Tipo di stampa Selezionare se effettuare una stampa/elaborazione di prova o definitiva.

Anno elenco / Numero elenco Confermare o modificare l'anno a cui si riferisce l'elenco e il numero dell'elenco da presentare, inteso come numero progressivo da presentare in dogana.

Varie

Primo elenco presentato Selezionare se è la prima volta che si presentano elenchi Intrastat.

Cessazione o variazione P.Iva Selezionare se è la prima volta che si presentano elenchi dopo aver cambiato la partita IVA o se l'azienda è in fase di cessazione.

I dati trimestrali si riferiscono al Se si tratta di elenco trimestrale, indicare se i dati si riferiscono al trimestre intero oppure, nei casi in cui si passa da regime trimestrale a mensile, e solo nel trimestre di transizione, se i dati si riferiscono solo al primo mese o ai primi due.

Questo campo, coi relativi valori "1° e 2° mese" o "1° mese", è utilizzato in fase di compilazione del file scambi.cee: esso è semplicemente un dato che è inserito nel frontespizio del file, subito dopo i dati relativi alla periodicità di presentazione.

Da notare che non può essere utilizzato per filtrare i movimenti da inserire in stampa o file, perchè sui movimenti non è indicata una data in cui il movimento dev'essere considerato, ma solo il mese o trimestre di competenza. Per filtrare i movimenti da inserire in stampa o file si può seguire una procedura specifica, ad esempio:
- impostare in "Anagrafica ditta - Dati Iva per anno - Periodo Intrastat" = "Trimestrale" ed estrarre i dati di tutto il trimestre;
- cancellare i movimenti Intrastat relativi al mese che non va considerato nel trimestre;
- impostare in "Anagrafica ditta - Dati Iva per anno - Periodo Intrastat" = "Mensile" ed estrarre i dati del mese.
Per la stampa, si reimposta "Trimestrale" e s'esegue la stampa; poi s'imposta di nuovo "Mensile" e s'esegue la stampa del mese rimanente.

Numero telefonico Inserire il proprio numero telefonico.

Partita IVA soggetto delegato In presenza di soggetto delegato, inserire la sua partita Iva.

Sezione doganale presentazione Inserire la codifica della Sezione doganale di competenza; viene recentata e sarà memorizzata sulle righe di movimenti intra presentate. Servirà a semplificare le registrazioni di rettifiche per i servizi.

Formato file Selezionare il formato del file ascii da produrre, tra "Intr@web", per la creazione del file "scambi.cee" da inviare tramite Intraweb, e "Telematico Interchange" per la creazione del file da inviare con Entratel, dotato di estensione ".I01".

Codice ZENT / data invio file / progressivo giornaliero invio file In caso si opti per un "Formato file" di tipo "Telematico Interchange", occorre compilare questi campi, necessari all'invio del file.

Nome da assegnare al file (senza path) Inserire il nome del file da generare, di norma denominato scambi.cee per il formato Intr@web. Per il formato Telematico Interchange, invece, il nome del file compare automaticamente una volta inseriti i campi Codice ZENT, data invio file, progressivo giornaliero invio file.
