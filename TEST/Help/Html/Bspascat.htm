Stampa controllo attività

Nome programma: BSPASCAT.DLL
Moduli richiesti: Parcellazione
Report programma: vai ai report

Questo programma esegue stampe di controllo sulle attività memorizzate nell'archivio, ad esempio per verificare eventuali attività prima di fatturarle, o per verificare che tutte le attività siano fatturate o che una singola attività sia attribuita a tutti i clienti interessati.

Considera data di esecuzione attività Abilita l'esecuzione di stampe da data a data esecuzione attività, permettendo quindi di abilitare o disabilitare i relativi campi.

Codice voce parcellazione Se indicato, stampa solo attività con codice parcellazione pari a quello impostato.

Da/A dipendente I due campi indicano un intervallo, da codice a codice dipendente, entro cui limitare la stampa delle attività.

Codice cliente Se impostato diverso da 0, stampa solo le attività assegnate ad un determinato cliente.

Da / A data inserimento modifica Quando si inserisce o modifica una voce di parcellazione, è memorizzata la data del giorno in cui è eseguita l'operazione. Tramite queste date è possibile filtrare i movimenti in base alla data effettiva di inserimento/modifica.

Nome dipendente Se impostato, visualizza solo i movimenti inseriti o modificati dal un dipendente, cioè nome operatore con cui si entra in Business.

Tipo riga Spuntare i tipi di attività da stampare.

Da contratto Selezionare se stampare solo le voci contrassegnate dal flag "Da contratto" o le non contrassegnate, oppure entrambe. Attualmente tale flag non attiva alcun automatismo, ma è solo un'informazione da gestire a mano.

Fatturato Selezionare se stampare solo le voci fatturate, solo le non fatturate o entrambe.

Emesso avviso di parcella Selezionare se stampare solo le voci per cui è stato emesso un avviso di parcella, oppure solo quelle per cui non è stato ancora emesso tale documento, o considerare entrambi i casi.
