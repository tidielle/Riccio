Esecuzione automatica ricezione E-mail

Nome programma: BNEMGMAI.DLL
Moduli richiesti: Business e-mail o CRM (in caso di CRM e-mail disponibile sulle sole funzionalità CRM)

Il programma di ricezione e-mail effettua la connessione POP3 al server di posta e scarica le nuove e-mail, solo dopo che è stata effettuata la ricezione le e-mail saranno disponibili nella console. Durante la ricezione viene alimentato il database con le informazioni relative al corpo e all'intestazione dell'e-mail, mentre gli allegati vengono salvati sulla cartella Asc\Allegati del server. I file sono codificati per impedirne l'accesso.

Le opzioni disponibile nel programma sono:

  • E-mail (0 = tutte): permette di scegliere se scaricare le e-mail di tutti gli operatori che hanno la posta configurata o specificare di scaricare le e-mail per un singolo indirizzo e-mail.
  • Scarica solo l'e-mail con ID: consente di scegliere di scaricare una singola e-mail del server. L'id che viene richiesto è quello che viene associato dal server di posta all'e-mail.
    Per gli account IMAP non è possibile scaricare una e-mail partendo dal suo ID, come avveniva per il POP3. Per risolvere il problema che impediva di generare il file di testo con l'e-mail, ora in caso di problemi il file viene generato in automatico, e viene indicata una e-mail nella cartella "archivio storico business" contenente il collegamento al file che potrà essere usato per trovare l'errore.
  • Genera file con l'e-mail non elaborata: quando si specifica di scaricare una singola e-mail, è possibile generare un file di testo contenente tutta l'e-mail senza che questa sia stata processata invece che scaricarla e salvarla sul database. Dalla consolle e-mail invece basta selezionare l'e-mail che è stata scaricata male e premere "CTRL+ALT+SHIFT+G" e nella directory di business verrà generato un file chiamato "BNEMGMAI_[indirizzo e-mail].txt" che il potrà permettere di replicare il problema.

Esecuzione in modalità batch

Il programma consente l'esecuzione in modalità batch, per scaricare le e-mail in automatico. Premendo Ctrl+Alt+F5 viene generato un file di estensione BUB con le impostazioni di esecuzione. Successivamente il programma provvederà a scaricare le nuove e-mail ad ogni intervallo di tempo se viene eseguito tramite un collegamento di business con i parametri: Operatore Password Azienda Profilo BNEMGMAI /B Percorso_del_file_bub. Esempio: Roby . PROVA . BNEMGMAI /B c:\business\bnemgmai.bub

La gestione degli errori

Se durante la ricezione delle e-mail si verificano dei problemi e il messaggio non viene scaricato, il programma segnala a video tramite un messaggio di errore o se è in esecuzione in modalità batch tramite file di log. Per evitare di dover controllare costantemente il file di log viene anche generata una e-mail nella cartella della posta in arrivo del destinatario del messaggio; questo messaggio riporta l'ID dell'e-mail che ha generato il problema, così da poter provare a scaricarla. Se il messaggio non viene comunque scaricato allora si può scegliere di generare il file con l'e-mail non elaborata e inviarlo all'assistenza per poter verificare il problema.
