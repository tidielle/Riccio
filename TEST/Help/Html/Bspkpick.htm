Gestione Picking

Nome programma: BSPKPICK.DLL
Moduli richiesti: Picking su Palmare
Opzioni programma: vai alle opzioni
Report programma: vai ai report

Questo programma permette di gestire le picking lists, in particolare è possibile:

  • creare o cancellare file di picking
  • visualizzare tutti i file di picking disponibili ed il loro contenuto
  • aprire i file di picking completati per acquisire le operazioni effettuate sul Pocket pc ed aggiornare le note di prelievo relative in modo che la quantità indicata sulle note sia uguale a quella effettivamente prelevata.

Modalità operative

Apri picking N.

Qualora si desideri aprire una picking list particolare già esistente per controllare il contenuto o cancellarla o acquisire le operazioni effettuate, selezionare questa voce ed indicare il numero della picking list da aprire.

Verrà visualizzata una griglia con l'elenco delle note di prelievo contenute nella picking ling indicata e sarà possibile:

  • 'File\Cancella Picking', per cancellare la picking list
  • 'File\Acquisisci', se il file associato alla picking list è già stato rinominato con '_OK' finale (il chè significa che dal teminalino è stato prelevato il materiale), verranno aggiornare le note di prelievo associate
  • 'File\Visualizza file di Picking', per visualizzare l'elenco del materiale da prelevare relativo alla picking list aperta

Apri Note di prelievo NON associate a picking

Qualora esistano delle note di prelievo non ancora associate a nessuna picking list, selezionando questa modalità, sarà possibile inserire una serie di filtri per identificare tali note di prelievo, ed in seguito generare le picking list.

In particolare, dopo aver inserito i filtri, verrà visualizzata una griglia con l'elenco delle note per cui non è stata associata una packing list. Mettendo la spunta nella colonna 'Seleziona' e cliccando dal menù 'File' la voce 'Genera Picking', verrà generata una picking list contenente tutte le note selezionate.

Apri tutti i picking

Qualora si desideri aprire più picking lists per controllare il contenuto o cancellarle o acquisire le operazioni effettuate, selezionare questa voce verrà aperta una finestra da cui è possibile selezionare le picking lists da aprire.

In particolare, nella griglia del programma, sono presenti due colonne 'Seleziona' e 'Merce prelevata'; la prima serve per selezionare le note di prelievo per cui generare le picking list, mentre la seconda viene abilitata automaticamente dal programma nel momento in cui dal terminalino viene completata la picking list e sta ad indicare che la merce è stata prelevata ed è possibile procedere con l'acquisizione del file elaborato.

Selezione dati Note di prelievo

Dalla data

alla data

Qualora si vogliano considerare solo le note di prelievo inserite in una determinata data (o di un range di date) indicare in questi due campi il primo e l'ultimo di essi.

Tipo bolla fattura

Se necessario, indicare il tipo BF che devono avere le note di prelievo da selzionare.

Magazzino

Qualora si vogliano selezionare le note di prelievo relative ad un magazzino particolare, indicare il codice specifico.

Causale magazzino

Qualora si vogliano selezionare le note di prelievo con una causale di magazzino specifica, indicare il codice.

Zona

Qualora si vogliano selezionare le note di prelievo relative ad una zona particolare, indicare il codice specifico.

Agente

Qualora si vogliano selezionare le note di prelievo relative ad un agente particolare, indicare il codice specifico.

Dalla provincia ... Alla provincia

Qualora si vogliano selezionare le note di prelievo di clienti di particolari province, indicare le province in questi campi.

Da vettore 1 ... A vettore 1

Qualora si vogliano selezionare le note di prelievo relative ad un dato vettore o ad un range di vettori, indicare i codici vettori.

Volume

Riporta la sommatoria della quantità per il campo 'volume' di anagrafica articolo di tutte le righe presenti nella nota di prelievo.

Pulsante Genera picking

Avvia la generazione del file di picking dalla note di prelievo selezionate.

Assegna vettore e memorizza ...

Permette di aggiornare tutte le note di prelievo collegate al file di picking che si sta creando assegnando un codice vettore (se diverso da zero).

Assegna e memorizza in 'num. partita' ...

Permette di aggiornare tutte le note di prelievo collegate al file di picking che si sta creando assegnando un numero di viaggio al codice partita (se diverso da zero).
