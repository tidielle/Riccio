Configurazioni per importare da file ASCII

Lanciando un profilo di import da file ASCII, per esempio lanciando il profilo 2

si potrebbe presentare il seguente errore:

OleDbException: -2147217911 - Impossibile aggiornare. Database o oggetto di sola lettura.

Questo perchè per l'import di file ASCII il motore di Business NET fa uso del motore Jet 4.0 e probabilmente l'estensione del file che si sta importando non è definita nel registro, nell'esempio indicato sopra del profilo 2 il file importato ha estensione .dat. Il "problema" si risolve configurando la chiave ad esempio da:

[HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Jet\4.0\Engines\Text]
"DisabledExtensions"="!txt,csv,tab,asc,tmp,htm,html"


NB: sulle nuove macchine a 64bit tale chiave potrebbe essere in [HKEY_LOCAL_MACHINE\SOFTWARE\Wow6432Node\Microsoft\Office\14.0\Access Connectivity Engine\Engines\Text] oppure [HKEY_LOCAL_MACHINE\SOFTWARE\Wow6432Node\Microsoft\Office\15.0\Access Connectivity Engine\Engines\Text] (si consiglia di cercare nel registro di windows tramite trova, la chiave DisabledExtensions).

a:

[HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Jet\4.0\Engines\Text]
"DisabledExtensions"="!txt,csv,tab,asc,tmp,htm,html,dat"
