Gestione Sconti per Classe Conto / Classe Articolo

Nome programma: BSVECLSC.DLL
Moduli richiesti: Vendite/fatturazione, Magazzino, Ordini, Magazzino Easy, CRM, Ordini Easy

Questo programma gestisce l'applicazione di sconti differenziati sia sulla base del Cliente che sulla base del prodotto acquistato dal Cliente stesso. Potendo suddividere i clienti in 999 Classi Sconto Cliente e gli articoli in altrettante 999 Classi Sconto Articolo, si possono creare numerosissime combinazioni in grado di gestire la maggior parte delle esigenze.

Ad es., supponiamo di avere 3 categorie di articoli (famiglia A, famiglia B, famiglia C) e 2 categorie di clienti (dettaglianti e grossisti) e d'impostare degli sconti differenziati come nella tabella seguente:

Famiglia A FamigliaB Famiglia C
Dettaglianti 10.0 + 3.0 15.0 8.0
Grossisti 10.0 + 9.0 22.0 + 3.0 15.0

Per suddividere gli articoli in classi e così attribuire diverse percentuali di sconto, è sufficiente:

- codificare le classi di sconto articoli 1=FAMIGLIA A -- 2=famiglia B -- 3=Famiglia C

- codificare le classi di sconto clienti 1=Dettaglianti -- 2=Grossisti

- attribuire agli articoli appartenenti alla categoria Fam. A, nel campo relativo alla Classe di Sconto Articolo dell'Anagrafica Articoli, il valore 1, agli articoli della Fam. B il valore 2 e a quelli della Fam. C il valore 3.

La stessa suddivisione può essere fatta anche per i clienti attraverso il campo Classe di Sconto Cliente nell'anagrafica Clienti, inserendovi il valore 1, ad esempio, per i dettaglianti e il valore 2 per i grossisti.

Attraverso il programma Gestione Sconti per Classi Conto/Classi Articolo, dopo aver selezionato la Classe Sconto Clienti che ci interessa, immettiamo i seguenti dati:

Per i dettaglianti (Codice Classe Sconto Cliente = 1)

Classe Articolo Sconto1 Sconto2 Sconto3 Da Data A Data
1 10.0 3.0 0 12/12/2016 12/12/2018
2 15.0 0 0 12/12/2016 12/12/2018
3 8.0 0 0 12/12/2016 12/12/2018

Per i grossisti (Codice Classe Sconto Cliente = 2)

Classe Articolo Sconto1 Sconto2 Sconto3 Da Data A Data
1 10.0 9.0 0 12/12/2016 12/12/2018
2 22.0 3.0 0 12/12/2016 12/12/2018
3 15.0 0 0 12/12/2016 12/12/2018

Da ora in avanti, se ci si trova nel periodo 12/12/2016 - 12/12/2018, all'atto dell'emissione di una bolla o di una fattura ad un cliente dettagliante, su una riga relativa ad un articolo della famiglia A, Business propone automaticamente uno sconto di riga del 10,0% + 3,0%.

Cl. Articolo Colonna della griglia in cui inserire il numero associato alla Classe Sconti Articoli a cui fa riferimento la combinazione.

Sconto 1 Inserire in questa colonna la prima percentuale di sconto.

Sconto 2 Inserire in questa colonna la seconda percentuale di sconto. Questa seconda percentuale di Sconto sarà scalata dopo aver scalato la prima.

Sconto 3 Inserire in questa colonna la terza percentuale di sconto. Questa terza percentuale di sconto sarà scalata dopo aver scalato le prime due.

Sconto 4 Inserire in questa colonna la quarta percentuale di sconto. Questa percentuale di sconto sarà scalata dopo aver scalato le prime tre.

Sconto 5 Inserire in questa colonna la quinta percentuale di sconto. Questa percentuale di sconto sarà scalata dopo aver scalato le prime quattro.

Sconto 6 Inserire in questa colonna la sesta percentuale di sconto. Questa percentuale di sconto sarà scalata dopo aver scalato le prime cinque.

Data aggiorn. Inserire in questa colonna la data d'inizio della validità di questa classe.

Data scad. Inserire in questa colonna la data di scadenza della validità di questa classe.
