Profilo 18 - Import catalogo fornitore

Questo profilo permette di eseguire l'importazione di un catalogo fornitori da file Excel.

Il file di excel da importare deve avere il seguente tracciato:

  • Codice
  • Descrizione
  • Descrizione2
  • UnMis
  • ClasseSc
  • MinOrd
  • Confez
  • Prezzo
  • Barcode

Definizione file Excel

Il file di Excel dovrà avere:

  1. Il nome del file dovrà essere: 'es_catalogo_semp.xls'.
  2. Il nome del primo foglio di nome: 'Catalogo'.

  3. Le intestazioni delle colonne ABCDEFGHI: nella riga 1 per ogni colonna si dovranno specificare il nome dei campi del tracciato record.
    N.B.: è importante rispettare l'ordine del tracciato.

  4. Definire la formattazione delle colonne come segue:

    ColonnaTracciatoFormattazionePosizioni decimali (per colonne numerica)
    ACodiceTesto-
    BDescrizioneTesto-
    CDescrizione2Testo-
    DUnMisTesto-
    EClasseScNumero0
    FMinOrdNumero3
    GConfezNumero3
    HPrezzoNumero2
    IBarcodeTesto-

    Per impostare la formattazione cliccare col pulsante destro del mouse nell'intestazione della colonna ABCDEFGHI e selezionare Formato Celle.

    Per formattare il Testo:

    Per formattare il numero selezionare Numero e le Posizioni decimali:

  5. Per l'inserimento dei dati nel file di Excel tenere conto dei seguenti limiti:

    ColonnaTracciatoFormattazioneDimensione caratteri digitabili
    ACodiceTesto30
    BDescrizioneTesto40
    CDescrizione2Testo40
    DUnMisTesto3
    EClasseScNumeroDa 0 a 999 con 0 decimali
    FMinOrdNumeroDa 0 a 999999999 con 3 decimali
    GConfezNumeroDa 0 a 999999999 con 3 decimali
    HPrezzoNumeroDa -9999999999 a 99999999999 con 2 decimali
    IBarcodeTesto18

  6. Lancio del profilo: il profilo al lancio chiederà dei dati comuni a tutti i record del file Excel in particolare:

    Il codice fornitore (inserire un valore numerico intero > 0)

    Il codice iva (inserire un valore numerico intero > 0)

    Il codice marca (inserire un valore numerico intero >= 0)

    La sigla fornitore (inserire un valore stringa non superiore di 10 caratteri)

  7. Tutti i dati comuni richiesti all'avvio dell'import ed i dati del file *.XLS verranno inseriti nei rispettivi campi della tabella ARTEST, inserendo i record non presenti e aggiornando quelli già inseriti.
